# BMW-SALES-ANALYSIS-2010-
An interactive dashboard that visualizes key trends, regional performances, top car models, and market opportunities. The dashboard leverages Power BI to provide insights into BMW's global sales performance in 2010.
## Data Source
Data obtained from Kaggle.[<img width="1170" height="434" alt="bmw latest dataset" src="https://github.com/user-attachments/assets/ff338321-a46a-47f1-9aa3-17bf3f07bd7d" />


## Project Overview
This project showcases a dynamic BMW sales dashboard built using Power BI. The raw dataset from Kaggle was cleaned and structured into a tabular format using Excel to facilitate analysis and visualization.

## Global Sales Performance
- **Sales Highlights:** BMW Group sold **$250M** globally in 2010.
- **Growth Drivers:** Strong growth observed in **Europe**, **Asia**, and **Africa**.
- **Top Series:** Top-selling series include **X5**, **3 Series**, and **X1**.

## Challenges and Solutions
- **Competition:** Intensifying rivalry from Tesla, Mercedes, and Chinese brands.
  - **Solution:** Differentiate through driving experience, M performance models, and digital services.
- **High Costs & Inflation:** Reduced purchasing power leading buyers toward used cars or delayed purchases.
  - **Solution:** Offer financing options or certified pre-owned vehicles.

## Conclusion
BMW's sales data reflects a strategic shift towards electrification, regional adaptation, and luxury segmentation. Leveraging data insights enables targeting growth markets and mitigating risks, allowing BMW to maintain its premium positioning and drive future growth.
